Expense Tracker:

A Java-based desktop application designed to efficiently track, manage, and organize daily expenses. The system provides a clean graphical interface with persistent database storage, enabling users to maintain accurate financial records with ease.

Key Features:

->Create, update, and delete expense records

->Categorize expenses for structured financial tracking

->Persistent data storage using database connectivity

->Intuitive and responsive desktop user interface

->Modular and maintainable project structure

Technology Stack:

->Java

->Java Swing (GUI)

->Maven (Build & Dependency Management)

->JDBC (Database Connectivity)

Project Overview:

The application is developed using Java Swing for the front-end interface and JDBC for backend data persistence. User interactions trigger business logic that performs CRUD operations on expense data stored in a relational database. Maven is used to manage dependencies and streamline the build process.

Installation and Execution:

1. Clone the repository:

   git clone https://github.com/sarikaa2020/Expense-Tracker.git
2. Open the project in a Java IDE (IntelliJ IDEA, Eclipse, or NetBeans)

3. Build the project:

   mvn clean install
4. Run the main application class from the IDE
